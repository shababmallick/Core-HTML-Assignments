  2. CSS

    3. Primary Use Cases Key Features/Benefits
    CSS uses selectors to target HTML elements and apply styles to them.This allows for precise control over how different parts of webpage are displayed.
    • CSS uses selectors to target HTML elements and apply styles to them. This allows for precise control over how different parts of a webpage are displayed.
    • The "C" in CSS stands for cascading, which means that styles can be inherited from parent elements or overridden by more specific styles.
    • CSS defines the box model, which governs how elements are spaced, sized, and positioned on a webpage.
    • CSS features like media queries allow for the creation of responsive layouts that adapt to different screen sizes and devices.
    • Modern CSS includes powerful layout tools like Flexbox and Grid that make it easier to create complex, responsive designs.

  3. JS

    4. Primary Use Cases Key Features/Benefits
    Building the structure of web pages
    • JavaScript is mainly used to add interactivity and dynamic behavior to websites, allowing for real-time updates and user interactions without the need to reload the page.
    • JavaScript is used to interact with and modify the Document Object Model (DOM), which represents the structure of a webpage. This enables the creation of dynamic and responsive web applications.
    • JavaScript can make requests to servers (using techniques like AJAX or Fetch API) to retrieve data and update parts of a webpage without a full page refresh.
    • JavaScript is often used to validate form inputs on the client side, providing instant feedback to users.
