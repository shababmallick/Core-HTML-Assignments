1. html - The root element of an HTML document, containing all other HTML elements.
  2. head -Contains metadata about the document, including the title, character encoding, and links to external resources like stylesheets.
  3. title -Sets the title of the web page, displayed in the browser's title bar or tab.
  4. meta - Provides metadata about the document, such as character encoding and viewport settings.
  5. link - Used to link external resources like stylesheets,icons, or alternate versions of the page.
  6. style - Contains CSS (Cascading Style Sheets) for styling the document's content.
  7. script - Embeds or links to JavaScript code for adding interactivity to the web page.
  8. body - Contains the visible content of the web page, including text, images, and other elements.
  9. Heading elements, ranging from h1 (highest level) to h6 (lowest level), used to define section headings and titles.
  10. p - Defines a paragraph of text, creating a new line and adding spacing between paragraphs.
  11. a - Creates hyperlinks to other web pages or resources, allowing users to navigate to different locations on the web.
  12. img - Embeds images in the document, displaying pictures and graphics on the web page.
